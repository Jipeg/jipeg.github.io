Алгоритмы поиска максимума в массиве

1. Поиск максимального элемента, если не нужен индекс этого элемента. Первые 2 строки кода генерируют массив из 10 элементов, заполненный случайными числами, которые меньше 100. Вместо этих строчек кода у вас может быть задан свой массив, например чтением из файла f как-то так: mas = f.readlines() 


import numpy as np
mas = np.random.randint(100, size=10)
print (mas)

maxi = 0
for i in mas:
    if maxi < i:
        maxi = i
print(maxi)

Как это работает? Всё начинается с выбора названия переменной, в которой будет ответ. Пусть это будет переменная переменной maxi. Сначала ей присваивается значение 0, так как я заранее знаю, что все числа положительные, а значит они заведомо больше нуля. Далее циклом for посредством переменной i перебираются все элементы массива. На каждой итерации цикла выполняется условие: если текущее значение maxi меньше очередного элемента i, то maxi = i. После завершения работы цикла в переменной maxi будет находиться ответ. Его просто выводим

Проблемы кода: 1) Надо знать, что числа положительные. 2) Хотелось бы избавиться от лишнего if. Решим эти проблемы следующим образом:


import numpy as np
mas = np.random.randint(100, size=10)
print (mas)

maxi = mas[0]
for i in mas:
    maxi = max(maxi, i)
print(maxi)

1) Теперь maxi инициализирована первым числом в массиве, а максимум, как известно, всяко не меньше этого первого числа. 2) Условие if удалось заменить на функцию max, которая возвращает максимальное из двух чисел

2. Поиск максимального элемента, если нужен индекс этого элемента. Первые 2 строки кода генерируют массив из 10 элементов, заполненный случайными числами, которые меньше 100. Вместо этих строчек кода у вас может быть задан свой массив, например чтением из файла f как-то так: mas = f.readlines() 


import numpy as np
mas = np.random.randint(100, size=10)
print (mas)

maxi = mas[0]
maxindex = 0
for i in range(0, len(mas)):
    if maxi < mas[i]:
        maxi = mas[i]
        maxindex = i
print(maxi, maxindex)

В этом алгоритме уже не обойтись без if-а внутри цикла, поскольку нужно менять две переменные. Изначально maxindex равен нулю на случай, если нулевой элемент окажется максимальным

3. Поиск максимального четного элемента, если не индекс этого элемента. Первые 2 строки кода генерируют массив из 10 элементов, заполненный случайными числами, которые меньше 100. Вместо этих строчек кода у вас может быть задан свой массив, например чтением из файла f как-то так: mas = f.readlines() 


import numpy as np
mas = np.random.randint(100, size=10)
print (mas)

maxi = mas[0]
for i in mas:
    if i % 2 == 0:
        maxi = max(maxi, i)
print(maxi)

Здесь внутри цикла обязательное условие проверки текущего элемента массива на делимость на 2. Только в случае выполнения этого условия элемент может претендовать на роль максимального

Упражнения:

  2. Найти максимальное число в файле task1.txt, которое делится на 5, но не делится на 15. Правильный ответ: 190
  3. *В этом же файле найти минимальное число, кратное 3 и 7

Алгоритмы поиска количества в массиве

1. Поиск количества четных элемента, если не нужны их индексы. Первые 2 строки кода генерируют массив из 10 элементов, заполненный случайными числами, которые меньше 100. Вместо этих строчек кода у вас может быть задан свой массив, например чтением из файла f как-то так: mas = f.readlines() 

        
import numpy as np
mas = np.random.randint(100, size=10)
print (mas)

kolvo = 0
for i in mas:
    if i % 2 == 0:
        kolvo += 1
print(kolvo)

Как и в случае максимума сначала выбираем переменную, в которой будем накапливать ответ. Пусть она называется kolvo. Изначально считаем, что элементов 0 (а вдруг в массиве нет четных числе!). В цикле проверяем условие кратности 2, если оно выполняется, то увеличиваем kolvo на 1.

2. Поиск количества четных элемента, если нужны их индексы. Первые 2 строки кода генерируют массив из 10 элементов, заполненный случайными числами, которые меньше 100. Вместо этих строчек кода у вас может быть задан свой массив, например чтением из файла f как-то так: mas = f.readlines() 

        
import numpy as np
mas = np.random.randint(100, size=10)
print (mas)

kolvo = 0
mas2 = []
for i in range(0,len(mas)):
    if mas[i] % 2 == 0:
        kolvo += 1
        mas2.append(i)
print(kolvo)
print(mas2)

Поскольку теперь нам нужно записывать индексы найденных чисел, то создаем пустой массив mas2, в котором будем хранить индексы четных элементов, и делаем цикл по индексам массива mas. Когда элемент массива mas[i] является четным, тогда увеличиваем kolvo и, главное, добавляем в mas2 индекс i

Упражнения:

  1. Найти все четные числа из файла task2.txt и вывести их в отсортированном порядке. Для этого их нужно записать в массив, а потом его отсортировать
  2. Найти в том же файле количество чисел, кратных 3, но не кратных 9. Ответ: 47
